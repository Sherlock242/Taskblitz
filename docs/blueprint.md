# **App Name**: Task Blitz

## Core Features:

- User Management: Allow admins to create and manage users with roles (Admin, Member) and basic information (name, email).
- Template Creation: Enable the creation of task templates; each template will contain a list of predefined tasks.
- Task Assignment: Implement a feature to assign a task template to a user; the system will create and assign the listed tasks from the template.
- Task Tracking: Create a simple dashboard where users can view tasks, their assignments, and statuses.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and efficiency.
- Background color: Light gray (#F5F5F5) for a clean, uncluttered interface.
- Accent color: Teal (#009688) to highlight interactive elements and important information.
- Font pairing: 'Inter' for body text (sans-serif), and 'PT Sans' for headers (sans-serif), ensures readability and modern look.
- Use clean, line-based icons from a consistent set to represent tasks and user roles.
- Use a card-based layout for tasks and templates to provide a clear visual structure.
- Incorporate subtle animations on task status changes for a polished feel.
